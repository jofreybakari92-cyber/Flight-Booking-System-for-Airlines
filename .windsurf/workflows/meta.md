# /meta - Framework Information

Access MAx Framework information and utilities with unified commands.

## Usage

```bash
# Explain concepts
/meta explain "lock-free queue"

# Show model cost breakdown
/meta eco

# Open model calculator
/meta model-calc

# Show formatting guidelines
/meta format

# List project templates
/meta templates
```

## Commands

### /meta explain
Explain concepts in plain language or summarize current session.

### /meta eco
Show model cost breakdown and eco usage summary.

### /meta model-calc
Open the model cost calculator.

### /meta format
Display code style and formatting guidelines.

### /meta templates
Show available project templates.

This replaces the old `/humanify`, `/eco`, `/model-calculator`, `/format`, and `/templates` commands.
